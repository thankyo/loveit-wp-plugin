Official integration plugin, for https://loveit.tips

== Description ==

Official integration plugin, for https://loveit.tips, which simply adds `Love It FAB` on every post at right bottom corner with an `iframe`.

== Installation ==

1. Upload `loveit_integration.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
